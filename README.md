# art-gallery

A standalone Next.js art gallery project for a private collection-style website.

## Features
- editorial hero section
- filterable collection grid
- artwork detail focus area
- FAQ accordion
- inquiry modal and confirmation flow
- responsive layout

## Stack
- Next.js
- React
- Framer Motion
- Lucide React

## Run locally
```bash
npm install
npm run dev
```

## Production notes
- uses remote Unsplash images
- currently uses in-file demo artwork data
- inquiry flow is UI-only for now and not connected to backend delivery

## Suggested next steps
- connect inquiry form to email or WhatsApp flow
- replace demo artworks with CMS or structured content source
- add custom branding, favicon, and social preview image
